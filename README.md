# initial setup

In pp-client app:
```
npm link
```

In browser-extension app:
```
npm link pp-client
```

# reload
After every change to pp-client
```
make
```
reaload chrome extension
(useful reloading app: https://chrome.google.com/webstore/detail/extensions-reloader/fimgfedafeadlieiabdeeaodndnlbhid)
